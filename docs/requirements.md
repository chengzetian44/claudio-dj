# 项目开发需求规格

## 项目名称
私人音乐电台（Claudio DJ）

## 项目定位
Windows 系统上运行的私人音乐电台软件。以 Claudio 为 AI DJ 人格，整合音乐播放、智能推荐、生活助手与语音交互。

## 核心功能

### 1. 音乐播放
- 基于 NeteaseCloudMusicApi 获取歌曲
- 单个 `<audio>` 组件播放
- 预取未来 10 秒音频
- 支持播放/暂停/下一首/音量控制

### 2. AI DJ 对话
- WebSocket 流式聊天
- AI 以 DJ Claudio 人格互动
- 可点歌、问天气、查日程、闲聊

### 3. 智能推荐
- 结合用户语料（taste.md + 听歌历史）
- Claude 语义理解生成推荐
- 考虑时间/天气/场景

### 4. 日期与计划
- 页面显示当前日期
- 飞书日历 hook 读取日程
- 定时推送早间简报

### 5. 日间/夜间模式
- CSS 变量切换主题
- 粒子背景颜色随主题变化

### 6. 粒子化视觉风格
- Canvas 动态粒子动画背景
- 粒子连线、颜色、运动可配置

### 7. 语音合成
- Fish TTS 生成播报
- MP3 缓存到 /tts/ 目录
- 用于早间简报、天气播报等

## 非功能需求

- 运行于 localhost:8080
- 纯 HTML/CSS/JS 前端，零框架依赖
- 离线优先 PWA
- SQLite 本地持久化
- 模块化后端，各层职责分明

## 扩展功能（Phase 2+）

- UPnP 智能家居控制
- 跨设备同步
- 飞书消息通知
- 天气 API 接入

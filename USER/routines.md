# Claudio 日常流程

## 工作日
- 07:00 起床，轻快音乐
- 07:30 早间简报（天气 + 日程 + 推荐歌单）
- 09:00 开始工作，背景电子音乐
- 12:00 午休，舒缓音乐
- 14:00 下午工作，独立摇滚
- 18:00 下班，放松音乐
- 21:00 晚间阅读，Ambient
- 23:00 睡前，钢琴独奏

## 周末
- 09:00 自然醒，慵懒爵士
- 10:00 周末简报
- 全天随机推荐，偏向轻松风格
- 22:00 午夜电台模式

## 特殊情境
- 运动时：高能量电子
- 下雨天：Lo-fi / 爵士
- 心情低落：治愈系 / 轻音乐
- 专注工作时：白噪音 / Ambient
